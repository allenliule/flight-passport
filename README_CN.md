飞行护照是一个**极简主义的无服务器飞行日志与 Web-GIS 追踪器**，旨在用绝对优雅的方式可视化个人航空历史。本项目完全基于**单文件架构**构建，彻底剥离了现代前端框架的沉重包袱与复杂的构建工具，回归到原生 JavaScript 与纯 CSS 编排的纯粹高效。

## 无摩擦的开发哲学

现代 Web-GIS 开发日益受到商业 API 网关的限制，开发者往往需要绑定信用卡并获取访问令牌，仅仅为了渲染一张基础底图。飞行护照打破了这种依赖链条。通过**直接调用公开的谷歌地图 XYZ 瓦片流**，该应用在**不需要任何 API 密钥或注册**的情况下，完美呈现了高保真的标准渲染图与原始混合卫星图像。这种方法确保了你的飞行日志具备**永久的便携性与绝对的隐私性**，并能在任何断网或隔离环境中独立运行。

## 制图学的双重性与测地线真相

飞行护照的核心技术叙事在于其**实时投影切换**功能，它直观地展现了弯曲地球与扁平数字屏幕之间固有的几何冲突：

**球面大圆模式**计算的是飞机在三维空间中的真实物理轨迹。它通过**测地线插值算法**，将球体表面上的最短距离投影到墨卡托画布上，从而形成了现实航空中标志性的优美弧线。

**平面墨卡托模式**则完全在屏幕空间坐标系下运行。它直接**线性化两点之间的经纬度跨度**，在投影平面上呈现出笔直的线条，呼应了传统纸质平面海图的经典结构美学。

当引擎处理相同城市对之间的多条重叠航线时，数学计算模块会**自动引入动态偏移曲线**，防止视觉线条冲突，从而在稠密的全球飞行网络中依然保持结构层级的清晰。

## 架构设计与本地自治

用户界面遵循严格的**本地优先数据模型**。你的完整飞行记录、机场指标和历史统计数据完全存储在**浏览器本地的 Web 存储空间**中。程序不会连接任何外部服务器，任何数据都不会离开你的本地机器。

为了与当代的暗黑主题界面保持视觉上的高度凝聚，地图引擎**将实时 CSS 矩阵滤镜直接作用于地图画布**。在渲染图模式下，瓦片会经历精确的色相旋转、反转和亮度调整，从而构建出极具科幻感的数字制图风格；当切换到卫星模式时，滤镜会瞬间解除，完整保留地球表面未受干扰的真实自然地貌。

## 即时部署

要启动你的个人日志，只需下载 `flight_passport.html` 文件并在任何现代浏览器中直接运行。应用程序将自动通过安全的静态内容分发网络拉取轻量级地图库，并立即初始化你的全局工作空间。
