**飞行护照 (Flight Passport)** 是一款专为飞行爱好者和旅行者打造的**单文件、纯前端**飞行轨迹记录工具。它拥有深色科幻风格的界面，帮助你在世界地图上可视化所有的飞行足迹。

无需后端、无需编译、无需安装，下载即用。

### ✨ 主要功能

*   **🗺️ 双地图图层**：可在 **Google 标准渲染图**（经暗色滤镜处理）和 **Google 卫星混合图** 之间无缝切换。
*   **📐 双投影模式**：
    *   **球面大圆 (Great Circle)**：模拟真实地球表面的弧线航线（越洋航线）。
    *   **平面墨卡托 (Mercator)**：屏幕上的直线投影（便于查看航向）。
*   **📊 详细数据统计**：自动计算总里程、航班数、执飞机场数、国家/地区数以及预估飞行时长。
*   **💾 本地存储**：所有数据保存在浏览器的 `localStorage` 中，刷新页面或关闭浏览器后数据不丢失。
*   **🌐 全球机场库**：内置 150+ 全球主要机场数据（含 IATA 代码、经纬度、城市）。

### 🚀 如何使用

1.  **下载**：克隆本仓库或直接下载 `flight passport.html` 文件。
2.  **打开**：双击文件，使用现代浏览器（推荐 Chrome、Edge、Firefox）打开。
3.  **联网**：首次加载需要联网以获取 Google 地图瓦片资源。
4.  **添加航线**：在左侧侧边栏输入机场 **IATA 三字码**（例如：`PVG` 代表上海，`LHR` 代表伦敦），即可开始记录。

### 🛠️ 技术栈

*   **前端**：原生 JavaScript (ES6)、HTML5、CSS3
*   **地图库**：https://leafletjs.com/
*   **地图源**：Google Maps XYZ 瓦片（基础调用无需 API Key）

### 📝 注意事项

*   **机场代码**：应用仅识别 3 位字母代码（IATA）。如需添加小众机场，可自行编辑 HTML 源码中的 `AIRPORTS` 对象。
*   **地图滤镜**：为了适配深色 UI，“标准渲染图”模式使用了 CSS `filter`（色相旋转、反相）进行处理，呈现独特的赛博朋克风格；“卫星图”模式则保留真实地貌。
